## 📌브랜치 전략

- **main** : 배포용 브랜치. 직접 커밋 금지, PR로만 변경.
- **develop** : 개발 통합 브랜치. 모든 기능 브랜치 작업이 머지됨.
- **feature/**: 기능 개발 브랜치. develop에서 분기하여 작업 후 완료 시 develop으로 PR.
